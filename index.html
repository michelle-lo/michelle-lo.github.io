What is CoffeeScript?

  • CoffeeScript is a language that gets compiled to JavaScript
  • .coffee files are compiled to .js files

Benefits of CoffeeScript?

  • CoffeeScript is much more readable and understandable than JavaScript
  • It adds “syntactic sugar” to JavaScript since it makes things easier to express
  • Its syntax is similar to python, so you can write "less code and do more"
  • It is also easier to maintain

Installing CoffeeScript

  • Download your version of Node JS based on your OS
  • npm install --save-dev coffeescript
    • Installs locally for a project
  • sudo npm install --global coffeescript
    • Installs globally to execute .coffee files anywhere

Command Line Usage

  • coffee -c FileName.coffee
    • Complies the CoffeeScript into JavaScript
  • coffee FileName.coffee
    • Runs the CoffeeScript in the terminal

Functions

  • Defined by the function name, an equal sign, a list of parameters, an arrow, and the function body
    •         foo = (x, y) ->
          x * y

        foo 3,7
  • Doesn’t need keyword return because every function returns the result of the last line of code
  • Can call functions without parentheses
  • Need parentheses if you have no arguments

Splat

  • A convenient way to accept multiple values as arguments to functions
  • var_name...
  • The other arguments passed will be stored in the variable as an array
    •         food = ["pasta", "sushi", "steak", "burgers"]
        eat = (first, second, rest...) ->
          console.log "my favorite food is #{first}"
          console.log "my second favorite food is #{second}"
          for item in rest
            console.log "i also like #{item}"
        eat food... #passes the whole array in the function

        # my favorite food is pasta
        # my second favorite food is sushi
        # i also like steak
        # i also like burgers

Arrays

  • There are several different ways to initialize and declare arrays
    •         arr = [0,1,2,3]
        arr = [
          0,
          1,
          2,
          3
        ]
        arr = [
          0, 1,

        ]

        arr = [

          2
          3
        ]

Ranges

  • Ranges aren’t a standard object type in JS so they get compiled down to arrays
  • range = [int..int]
    • The values are inclusive
  • Can use them to access sections of array or create arrays
    •         days = [1..7] #[1,2,3,4,5,6,7]

        #makes an array with the elements of the days array
        #starting from index 1 to index 4
        two_to_five = days[1..4] #[2,3,4,5]

        #replaces a section of the array
        days[1..4] = ["two","three","four","five"] #[1,"two","three","four","five",6,7]
