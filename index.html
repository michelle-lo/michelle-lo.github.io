What is CoffeeScript?

  • CoffeeScript is a language that gets compiled to JavaScript
  • .coffee files are compiled to .js files

Benefits of CoffeeScript?

  • CoffeeScript is much more readable and understandable than JavaScript
  • It adds “syntactic sugar” to JavaScript since it makes things easier to express
  • Its syntax is similar to python, so you can write "less code and do more"
  • It is also easier to maintain

Installing CoffeeScript

  • Download your version of Node JS based on your OS
  • npm install --save-dev coffeescript
    • Installs locally for a project
  • sudo npm install --global coffeescript
    • Installs globally to execute .coffee files anywhere

Command Line Usage

  • coffee -c FileName.coffee
    • Complies the CoffeeScript into JavaScript
  • coffee FileName.coffee
    • Runs the CoffeeScript in the terminal
